# AI_Research

put some stuff - all data for the project

## About
This project investigates whether generative AI tools are a boon or bane 
for users in different fields.

## Folder Structure
- 01_Literature_Review: Journal articles, conference papers, books and news articles
- 02_Quantitative_Analysis: Survey data, survey questions, analysis scripts and reports
- 03_Qualitative_Analysis: Interview transcripts, protocols, consent forms and analysis reports
- 04_Drafts_and_Reports: Research proposals, conference papers and final reports
- 05_Additional_Materials: Information sheets, photos and media files
